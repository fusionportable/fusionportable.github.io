FusionPortable V2-Dataset Details

Organization¶

Note:

1. <frame_id>.yaml store intrinsics and extrinsics of a specific sensor
2. <map_env>.pcd ground-truth map
3. <platform_env>.bag rosbag
4. <platform_env>.7z compressed rosbag
